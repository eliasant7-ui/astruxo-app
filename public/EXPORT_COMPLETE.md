## 🚀 Quick Start Guide

### 1. Extract the Archive
```bash
tar -xzf ASTRUXO_COMPLETE_SOURCE_CODE.tar.gz
cd astruXo
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Configure Environment
```bash
cp env.example .env
# Edit .env with your credentials:
# - Firebase credentials
# - Agora App ID & Certificate
# - Stripe API keys
# - Database connection
```

### 4. Setup Database
```bash
npm run db:generate
npm run db:migrate
```

### 5. Start Development
```bash
npm run dev
```

Your platform will be running at `http://localhost:5173`
